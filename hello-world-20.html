Hello World 2.0!

August 22, 2017
  • blog
  • meta
  • pelican

So this blog did not receive any love from me for quite some time. That's why I decided to revamp the whole thing and now it looks completely the same! Want to read me ramble about that topic for some paragraphs? Then read on...

That one time I tried to run a blog

In the beginning, there was light. After that - in mid 2014 I think - I started this blog. And behold, I saw that all was well. I used Octopress, a Content Management Framework written in Ruby (Version 3 is promised to be coming since January 2015, so let's see how that will turn out). Octopress has nice tools for syntax highlighting and is basically a neat little static site generator extended by some blogging features.

But since Octopress is build in Ruby, a language that I have a love-hate relationship with, it soon came to problems. Ruby sucks (or at least it sucked back in the day) when it comes to managing packages. Back in 2013/2014 I did a lot of stuff with Ruby and Rails and every few weeks, some gem broke and I spend hours trying to fix that so I could continue to work.

The same thing happened with this blog. I could neither get Octopress to run, nor could I bring myself to debug it, mostly because I knew it would definitly break again.

Usually, the main apology for someone not blogging is the lack of time to write posts. But I was quite often willing to blog - I even set up a Wordpress blog. But the usual blogging systems are not right for me as they have all these features I don't use and are hard to customize.

Riding the Pelican

Now I took some time to set up a Pelican blog. Which is basically the same thing, but with Python. It took some hours to get the old theme from Octopress running, but now everything (except comments and some minor other stuff) is working.

And I have to say, I'm quite happy about it. The only thing left is writing some blog posts. Let's see how this turns out...
